## Deployment<br />


Please add an env.local file in the root, and add these values:

NEXT_PUBLIC_API_KEY=YOUR_API_KEY
NEXT_PUBLIC_ENDPOINT_URL=https://weather.visualcrossing.com/VisualCrossingWebServices/rest/services/timeline/
